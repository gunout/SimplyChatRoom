# SimplyChatRoom

Simple Chat Room 

*STRUCTURE DU PROJET :

    /votre-projet
    │
    ├── /public
    │   └── index.html  (votre fichier HTML)
    │
    └── index.js  (votre fichier contenant le code du serveur)


1 . INSTALLEZ LES DEPENDANCES
    
    sudo apt update
    sudo apt install nodejs npm
    npm init -y
    npm install express socket.io


2 . RUN     

    node server.js
